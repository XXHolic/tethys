# CSS

## 是什么

- https://developer.mozilla.org/zh-CN/docs/Web/CSS

### 1、指层叠样式表 (Cascading Style Sheets)

### 2、是一种 样式表语言，用来描述 HTML 或 XML（包括如 SVG、MathML、XHTML 之类的 XML 分支语言）文档元素的呈现

### 3、CSS 描述了在屏幕、纸质、音频等其它媒体上的元素应该如何被渲染的问题。
